1、该例程为uCOSII2.91+lwip-1.3.2 (DM9000A)使用例程。

2、使用说明
   (1)工程文件路径：例程目录\uCOSII2.91+lwip-1.3.2 (DM9000A)\MDK-ARM\Project.uvproj。
   (2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
   (3)下载调试工具为ULINK。
   (4)请将JP4插上跳线帽。
   (5)请将配套LCD模块插上板子CN13。
   (6)请将以太网双绞线连接到板子CN12。
   (7)开发板IP地址为192.168.0.100。
   (8)HEX文件下载到板子后,LED灯闪烁,LCD模块显示演示画面,打开网页输入IP地址,显示网页信息,表明例程运行正确。 
 
3、注意事项
   请务必在下载、调试、运行过程中，保持板子上电、ULINK连接并插在电脑上。
