1、该例程为Mouse (SL811 HY35A)使用例程。

2、使用说明
   (1)工程文件路径：例程目录\Mouse (SL811 HY35A)\MDK-ARM\Project.uvproj。
   (2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
   (3)下载调试工具为ULINK。
   (4)请将USB Mouse插在板子CN6上。
   (8)请将配套LCD模块插上板子CN13。 
   (9)HEX文件下载到板子后,LCD模块显示演示画面,表明例程运行正确。
 
3、注意事项
   请务必在下载、调试、运行过程中，保持板子上电、ULINK连接并插在电脑上。
