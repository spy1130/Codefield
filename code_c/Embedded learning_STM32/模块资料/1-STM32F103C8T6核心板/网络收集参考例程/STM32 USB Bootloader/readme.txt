1、该例程为STM32 USB Bootloader使用例程。

2、使用说明
   (1)工程文件路径：例程目录\STM32 USB Bootloader\MDK-ARM\Project.uvproj。
   (2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
   (3)下载调试工具为ULINK。
   (4)请将USB线插在板子。
   (5)HEX文件下载到板子后,打开我的电脑新识别磁盘,删除文件firmware.bin,往磁盘拷贝文件(LED1为读写指示灯),成功拷贝表明例程运行正确。
 
3、注意事项
   请务必在下载、调试、运行过程中，保持板子上电、ULINK连接并插在电脑上。
