1、该例程为Oscillograph (HY35A)例程。

2、使用说明
   (1)工程文件路径：例程目录\Oscillograph (HY35A)\MDK-ARM\Project.uvproj。
   (2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
   (3)下载调试工具为ULINK。
   (4)请将配套LCD模块插上板子CN13。
   (5)HEX文件下载到板子后,LED闪烁,LCD模块显示演示画面,表明例程正确运行。

3、操作提示
   TAMPER键 = 切换通道焦点(CH1或CH2)
   WAKEUP键 = 显示帮助或退出帮助
   USER  键 = 暂停或实时运行
   摇杆上键 = 放大波形垂直幅度或向上移动
   摇杆下键 = 缩小波形垂直幅度或向下移动
   摇杆左键 = 水平展宽波形
   摇杆右键 = 水平缩小波形
   摇杆OK键 = 切换摇杆调节模式(幅度或位置)

4、注意事项
   请务必在下载、调试、运行过程中，保持板子上电、ULINK连接并插在电脑上。
