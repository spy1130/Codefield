1、该例程为MP3 Player例程。

2、使用说明
    (1)工程文件路径：例程目录\MP3 Player\MDK-ARM\Project.uvproj
	(2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
    (3)使用调试工具为ULINK。
	(4)使用VS1003硬解MP3。
	(5)在SD卡根目录下创建文件夹SYSTEM,拷贝<汉字库.bin>至该目录,在SD卡根目录下创建文件夹Music,拷贝MP3文件至该目录。
	(6)请将JP3插1-2位置。
    (7)将SD Card插入SD卡座。
    (8)请将配套LCD模块插上板子CN13。
    (9)JOY_UP快进、JOY_DOWN快退、JOY_LEFT向上选择歌曲、JOY_RIGHT向下选择歌曲。
    (10)HEX文件下载到板子后,LED灯闪烁,LCD模块显示MP3播放界面。 

3、注意事项
	请务必在下载、调试、运行过程中，保持板子上电、ULINK连接并插在电脑机箱上。
