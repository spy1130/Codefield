1、该例程为VS1003(record)例程。

2、使用说明
   (1)工程文件路径：例程目录\VS1003(record)\MDK-ARM\Project.uvproj。
   (2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
   (3)下载调试工具为ULINK。
   (4)请将SD Card插入SD卡座。
   (5)SD卡中的voice.wav为录音文件。
   (6)Wakeup Button为开始录音键、User Button结束录音键、Tamper Button播放录音键。
   (7)操作步骤:⑴ JP3不插跳线帽,按下开始录音键。⑵ 按下结束录音键。⑶ JP3插1-2位置,按下播放录音键。
 
3、注意事项
   请务必在下载、调试、运行过程中，保持板子上电、ULINK连接并插在电脑上。
