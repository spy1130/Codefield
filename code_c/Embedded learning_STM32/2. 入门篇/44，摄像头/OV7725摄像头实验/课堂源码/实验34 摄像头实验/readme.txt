/**
 ***************************************************************************************************
 * 实验简介
 * 实验名称：摄像头 实验
 * 实验平台：正点原子 战舰F103开发板
 * 实验目的：学习OV7725摄像头模块的使用方法

 ***************************************************************************************************
 * 硬件资源及引脚分配
 * 1 LED灯
     DS0（RED）     : LED0 - PB5
     DS1（GREEN）   : LED1 - PE5
 * 2 独立按键
     KEY0    - PE4
     KEY1    - PE3
     KEY2    - PE2
     KEY_UP  - PA0  (程序中的宏名:WK_UP)
 * 3 串口1 (PA9/PA10连接在板载USB转串口芯片CH340上面)
 * 4 正点原子 2.8/3.5/4.3/7/10寸TFTLCD模块(仅限MCU屏，16位8080并口驱动)
 * 5 TPAD电容触摸按键(右下角LOGO,即TPAD,连接在PA1)
 * 6 外部中断8(PA8,用于检测OV7725的帧信号)
 * 7 定时器6(用于打印摄像头帧率)
 * 8 ALIENTEK OV7725摄像头模块,连接关系为:
        OV7725模块 ----------- STM32开发板
        OV_D0~D7  -----------   PC0~7
        OV_SCL    ------------  PD3
        OV_SDA    ------------  PG13
        OV_VSYNC  ------------  PA8
        FIFO_RRST -----------   PG14
        FIFO_OE   -----------   PG15
        FIFO_WRST ------------  PD6
        FIFO_WEN  ------------  PB3
        FIFO_RCLK ------------  PB4

 ***************************************************************************************************
 * 实验现象
 * 1 本实验开机后，先初始化摄像头模块，OV7725初始化成功，则在需要先按KEY0\KEY1选择OV7725为
    QVGA\VGA模式输出。我们可以通过KEY0设置光照模式（5种模式）、通过KEY1设置色饱和度，通过
    WK_UP设置对比度，通过TPAD设置特效（总共7种特效）。通过串口，我们可以查看当前的帧率
    （这里是指LCD显示的帧率，而不是指OV7725的输出帧率），DS0指示程序运行状态。同时可以借助
    USMART设置OV7725的寄存器，方便大家调试。调用SCCB_RD_Reg和SCCB_WR_Reg等函数,实现OV7725寄存器读写.

    OV7725_QVGA模式：传感器输出320*240分辨率输出至LCD，可视角度广，但画面没那么清晰和细腻。
    OV7725_VGA模式：将640*480分辨率截取中间的320*240输出至LCD，可视角度窄，但画面清晰和细腻。
 * 2 LED0闪烁 ,提示程序运行。

 ***************************************************************************************************
 * 注意事项
 * 1 4.3寸和7寸屏需要比较大电流,USB供电可能不足,请用外部电源适配器(推荐外接12V 1A电源).
 * 2 本实验需要自备ALIENTEK OV7725摄像头模块一个
 * 3 安装检查好摄像头再上电，注意不要插反
 * 4 AC6不开优化和Optimize for Time，否则可能无法正常运行
