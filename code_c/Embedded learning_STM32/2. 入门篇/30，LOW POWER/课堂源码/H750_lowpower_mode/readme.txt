 * 实验名称 : 待机模式 实验
 * 实验平台 : 正点原子 Mini Pro H750开发板
 * 实验目的 : 学习STM32待机模式的使用

 ***************************************************************************************************
 * 硬件资源及引脚分配
 * 1 RGB灯
     RED   : LED0 - PB4
 * 2 独立按键
     KEY0  - PA1
     WK_UP - PA0
 * 3 电池管理(低功耗模式 - 待机模式)
 * 4 ALIENTEK 2.8/3.5/4.3/7/10寸TFTLCD模块(仅限MCU屏，16位8080并口驱动)

 ***************************************************************************************************
 * 实验现象
 * 1 下载代码后，可以看见LED0闪烁，说明代码正在运行。
 * 2 按下按键KEY0后，TFTLCD屏熄灭，提示进入待机模式，此时LED0不再闪烁，说明已经进入待机模式。
 * 3 按下按键WK_UP后，TFTLCD屏点亮，提示退出待机模式，此时LED0继续闪烁，说明已经退出待机模式。

 ***************************************************************************************************
 * 注意事项
 * 1 在待机模式下，下载器无法下载程序，需要唤醒后才能下载
 * 2 唤醒后系统会进行复位，程序从头开始执行
