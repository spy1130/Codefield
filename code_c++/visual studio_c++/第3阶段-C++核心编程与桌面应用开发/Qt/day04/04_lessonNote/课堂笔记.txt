1 QFile文件读写
	通过 QFile file(文件名)创建一个QFile文件对象
	使用open函数来指定打开的模式，只读、只写、读写、追加、截断，都有相应的枚举值，追加和截断用在有写的模式
	readAll来读取文件所有的内容
	readLine 来读取单行  ，循环读取的话，判断是否到达文件结尾，atEnd来判断
	
	编码问题
		默认情况下Qt支持UTF 8   ， 如果读取其他编码，要将这些文件的内容进行转码
		QTextCodec *code  = QTextCodec ::codecForName("gbk") 获取一个codec对象
		codec -> toUnicode(array)  将某个数组转化成unicode的qstring
		
		写文件通过 write函数实现
		
2 QFileInfo 读取文件的额外信息
	QFileInfo info(文件路径名) 
	info.相应信息函数  获取文件的额外信息
	
	
	
3 翻金币案例
	需求分析：
		窗口标题、icon
		窗口大小固定
		菜单栏 - 退出
		
		三个场景：
		开始
			开始按钮：点击之后进入第二个场景  （第一天作业的知识，窗口切换），动画   自定义按钮实现
		关卡选择场景
			20个按钮：点击后都会进入第三个场景，第三个场景的硬币盘面不同   自定义按钮实现，paintEvent实现
			
		游戏场景
			4*4的宫格，放置16个硬币，点击一个硬币翻转，同时翻转上下左右4个硬币（如果有的画）
				金币变成银币，银币变成金币
				
			硬币也是按钮，点击后翻转动画     （自定义按钮）
			
			胜利条件：全部都是金币的时候游戏胜利，同时播放动画和音效
			
			
			
			
			
4 主窗口
	创建一个父类MyMainWindow ，作为基类，设定公有的参数
	创建其他窗口子类继承自MyMainWindow ，就可以不用设置这些参数
	
	背景图
		paintEvent 直接绘制 pixmap
		
	开始按钮 （自定义按钮）
		图片通过paintEvent绘制
		动画通过 QPropertyAnimation  属性动画去产生过度动画
			1位置大小属性发生变化
			2给定开始的位置大小属性
			3给定结束的位置大小属性
			4给定速度，给定动画时长
